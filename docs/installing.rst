############
 Installing
############

To install the package, you can use the following command:

.. code:: bash

   pip install anemoi-models

**************
 Contributing
**************

.. code:: bash

   git clone ...
   cd anemoi-models
   pip install .[dev]
   pip install -r docs/requirements.txt

You may also have to install pandoc on MacOS:

.. code:: bash

   brew install pandoc
